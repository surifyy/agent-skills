# TRTC Call

> Audio/video calling SDK supporting 1v1 and group calls with
> call invitation, accept, reject, and hangup management.

## Core Capabilities

- **1v1 Calling**: Voice and video calls between two users
- **Group Calling**: Multi-party audio/video calls
- **Call Management**: Invite, accept, reject, cancel, hangup, timeout handling

## Related Products

- RTC Engine powers the underlying audio/video transport
- Chat SDK provides the signaling channel for call invitations
