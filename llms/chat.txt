# TRTC Chat

> Instant messaging SDK providing message send/receive, conversation management,
> group chat, user profiles, and offline push notifications.

## Core Capabilities

- **Foundation**: SDK initialization, login, multi-device management, event listeners
- **Message**: Send, receive, history, recall, read receipts, offline push
- **Conversation**: Conversation list, unread count, pinning, drafts
- **Group**: Group creation, join/leave, management, group messaging, AVChatRoom
- **User & Relations**: User profiles, online status, friend management
