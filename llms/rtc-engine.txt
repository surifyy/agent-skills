# TRTC RTC Engine

> Core real-time audio/video engine providing room-based communication
> with stream publishing, subscribing, and mixing capabilities.

## Core Capabilities

- **Room Management**: Enter/exit room, room events, connection state
- **Publishing**: Camera/microphone capture, stream publishing, encoding parameters
- **Subscribing**: Remote stream subscription, video rendering, audio playback
- **Mixing**: Server-side stream mixing and transcoding
