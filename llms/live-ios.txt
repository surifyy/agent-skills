# TRTC Live — iOS

> iOS integration guide for TRTC Live SDK.
> Requires iOS 13.0+, Xcode 15+, CocoaPods.

## Quick Start

- [Run Demo App](https://trtc.io/document/60455.md) — Clone, configure, and run the official TUILiveKit demo
  - Repo: https://github.com/Tencent-RTC/TUIKit_iOS.git
  - Path: `TUIKit_iOS/application`
  - Setup: `pod install --repo-update` → edit `GenerateTestUserSig.swift` with SDKAppID + SecretKey
  - Requires: iOS 13.0+ real device, Xcode 15+, CocoaPods 1.10+
- [Build Live Streaming](https://trtc.io/document/74594.md)
- [Build Live Room](https://trtc.io/document/78279.md)
